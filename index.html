Web이란 무엇인가?

Website (English is webpage ): is a collection of text, images, document files suitable for the World Wide Web and executed in a web browser . A browser displays a web page on a computer screen or mobile device . The web page, often referred to as a file, is written in HTML or similar markup languages . Web browsers contain many web resource elements such as: CSS , client-side scripts , and images , intended to render a web page. The website is often misunderstood with a website , but in fact, a Webpage is just a certain part of the website
